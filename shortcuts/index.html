Keyboard Key Calculator Button Action
0 to 9 0 to 9 Appends single number to entry
n ± Negates the displayed number
. . Appends decimal point to entry
Backspace ← Deletes single number entry
Delete Clear Entry Resets the displayed number to 0
Esc Clear All Clears number entry and pending operations
% mod Applies the modulus operator
/ ÷ Applies the division operator
* × Applies the multiplication operator
- - Applies the subtraction operator
+ + Applies the addition operator
w √ Applies square root to the displayed number
q x² Squares the displayed number
r 1/x Applies reciprocal to the displayed number
"Enter" or "=" = Evaluates the equation
